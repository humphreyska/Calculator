Working Calculator

Co-authored by Kaitlyn Humphreys and Juliana Luczynski

This program implements a calculator which takes in positive and negative doubles and operands and then evaluates and displays the result.

How to Install Java:

Step 1: Go to Java SE - Download site

Step 2: Under Java Platform Standard Edition click download Oracle JDK

Step 3: Choose the JDK for your operating system (Windows, Mac, Linux)

Step 4: Accept licensing agreement and download

How to Install Eclipse:

Step 1: Make sure you have already installed java

Step 2: Go to Eclipse IDE for Java Developers

Step 3: Choose the download link for your platform (Windows, Mac, Linux)

Make sure it is still the Java Developers version.

Window Users:

1. Unzip the download file into a directory of your choice

Mac Users:

1. Double click the download image (DMG file)

2. Follow the on-screen instructions to download

3. Eclipse will be installed under /Applications/eclipse directory

Linux:

1. Install eclipse under /usr/local

2. To run: open /usr/local/eclipse click on the “Eclipse icon” or to start a terminal enter “eclipse”

If the download doesn't work, under more downloads Luna or Oxygen are good substitutes.

Download Calculator:

Step1: Github link

Click “src” to view individual files

Step 2: Select Working-Calculator.zip

Step 3: Click “view raw” to download

Step 4: Unzip the file

Step 5: Run Eclipse

Step 6: Click the “File” tab

Step 7: Click “Open projects from file system”

Step 8: Select your root directory, browse, choose the directory your stored Calculator-master

Step 9: Select all and finish

Step 10: You have successfully downloaded the Working-Calculator program

Run Calculator:

Step 1: Run Eclipse

Step 2: Right click Working Calculator

Select “run as”, Select “java application”

Step 3: Make window bigger to view entire application

Step 4: Type expresion into text box

For addition use “ + ”

For subtraction use “ - ”

For multiplication use “ * “

For division use “ / ”

For parenthesis use “ ( “ and “ ) ”

For numbers type any negative, positive, or decimal numbers

Step 5: Click calculate to evaluate the expression

The result will appear below text box

Step 6: Click clear to reset text box

Enter a new expression
